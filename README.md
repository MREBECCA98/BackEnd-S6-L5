Gestione prenotazioni Hotel - Web Application 

Questa applicazione web o stata sviluppata per la gestione delle prenotazioni di un hotel, permettendo al personale autorizzato di gestire clienti, camere e prenotazioni.
Il progetto utilizza ASP.NET Core MVC per la gestione dell'interfaccia utente e Entity Framework Core per l'accesso ai dati.
L'applicazione implementa autenticazione e autorizzazione basata sui ruoli tramite Identity
